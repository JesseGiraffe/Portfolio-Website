Folding Hanger Project

The Problem

During this project we were tasked with using the SOLIDWORKS sheet metal functions to design an object to improve our life. My dorm room has a clothing rack that was installed too close to the wall causing my shirts to tilt upwards from contacting the back wall. I felt this project would be a great way to fix that issue while also demonstrating my mechanical design skills.

The Process

I first designed the two halves of the hanger using the SOLIDWORKS sheet metal functions. The dimensions were based off standard sheet metal clothing hangers I already owned. Then I implemented the central pin making sure the tolerances were just tight enough to allow intended movement of the mechanism without anything unexpected. A spacer disk was implemented between the two halves of the hanger and cutouts were made in either end so that they can rotate smoothly around the end cap. It is designed in a way so when weight is placed onto the hanger the downwards pressure helps to impede movement by locking the components in place. Lastly a screw on top was added connecting the actual hanger hook to the rest of the body

The Results

A simple yet elegant foldable hanger designed with manufacturability and practicality in mind. I hope to use some of the shop equipment to produce a real version of my CAD design soon. This project demonstrates my skills using the SOLIDWORKS sheet metal functions and design of mechanical mechanisms.
