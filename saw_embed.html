Saw Handle Project

The Problem

This project was all about ergonomics we were tasked with redesigning an existing tool handle of our own choice. I selected a saw handle for my redesign intending to make it more usable for both people with larger hands and those in cold environments wearing gloves while working.

The Process

Once a tool handle had been selected, I began on creating several concept sketches. We then developed these sketches into several clay models to gather user feedback and thoughts. Through these user interaction tests we were able to further refine our designs, maximizing both comfort and useability. We used this gathered data to produce a series of hard foam models each of which further refined the design until we created our final painted model witch we mounted on the actual tools in my case a saw blade.
